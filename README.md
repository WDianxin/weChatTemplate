# 分支

  + dev : 测试分支

# 数据结构

```shellg
|- commponents                            自定义组件库
		|- listOrderDf 待服务订单
		|- listOrderDg 待处理订单
		|- listOrderYw 已完成订单
		|- navbar      自定义头部导航栏
		|- region      注册
		|- searchshow  地区选择
		|- swiperComp  轮播图
		|- typeWash    首页清洗类型 权限
|- image                                  静态图片
|- page                                   整体页面
|- requestAPI                             网络请求封装
|- template                               模板
|- utils                                  工具集合
|- vant-weap                              有赞库
|- app.js                                 页面入口js
|- app.json                               页面注册和功能设置
|- app.wxss                               公共css
|- package.json                            有赞版本

```

# 页面统计
 ```shellg
|- 首页:pages/index/index.wxml
|- 登录页:pages/login/login
|- 订单:pages/order/order.wxml 
|- 我的:pages/my/my.wxml
|- 我的资料:pages/myData/myData.wxml
|- 修改密码认证:-pages/changePassword/changePassword.wxml
|- 密码修改:pages/newPassword/newPassword.wxml
|- 认证成功/失败：pages/identitySuccess/identitySuccess
|- <!--pages/options/options.wxml  设置-->
|- <!--pages/listOrders/listOrders.wxml 我的订单-->
|- <!--pages/searchListOrders/searchListOrders.wxml 我的订单搜索-->
|- <!--pages/changeMobile/changeMobile.wxml 换绑手机号-->
|- <!--pages/forgotPassword/forgotPassword.wxml 忘记密码-->
|- <!--pages/swiperCompH5/swiperCompH5.wxml 轮播图外联h5-->
|- <!--pages/shareAppMessage/shareAppMessage.wxml 分享页面-->
```
